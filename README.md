# Stratifyd Public API Website

This is the website for public access.

Host List

1. [Github](https://stratifyd.github.io/APIWebSite/)


## Setup 

```shell
bundle install --path vendor/bundle
```

## Deploy

### Github

1. Commit changes
2. Run `./deploy.sh`

### S3

1. Run `./upload.sh {bucket_name}`
